# VectorPy
Python version of Seanthebuilder1's VectorMath

# Compile
To compile using g++ use command:
```
g++ -O3 -std=c++17 -static-libgcc -static-libstdc++ -o main.exe -g ./src/*.h ./src/*.cpp
```

# Using CMake
To compile using CMake
```
cmake . -G "MinGW Makefiles"
make
```
